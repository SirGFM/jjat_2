About the game

"JJAT+" is a co-op game in which two robots must help each other through their adventure. Play it with a friend or try to control both characters at once.

Launching the game

The preferred way to download and launch the game is through itch, the itch.io app.

The app automatically manages updating the game and has built-in launch options.

If you manually downloaded the game, there are a few ways to easily launch the game in one of the specified modes:

Windows

There are three batch scripts that handle launching the game:

  • Single_Player_Mode.bat: Launch the game in single player mode.
  • Two_Players_Mode.bat: Launch the game in two players mode.
  • Synchronous_Single_Player_Mode.bat: Launch the game in synchronous single player mode.

Simply double-click the desired script.

Linux

There's a single script used to launch the game, launch.sh. Execute that script with one of the following arguments:

  • --single-player-mode | -1p: Launch the game in single player mode.
  • --two-player-mode | -2p: Launch the game in two players mode.
  • --synchronous-single-player-mode | -s1p: Launch the game in synchronous single player mode.

How to play

Controls

Currently, each character is assigned four actions: move left, move right, jump and action.

The following schemes are present, and accessible from the launch game menu (if launching from the itch.io app):

Although not yet implemented, the pause action is mapped to the first two gamepad's pause button (if connected) and to the Escape key.

Single player mode

Control both characters at once. If using a keyboard (it must have two analog sticks and four should buttons), the left half of the controller controls Red, while the right half controls Blue.

On the keyboard, Red is controlled with the WASD (or ZQSD) keys and Blue is controlled with the arrow keys

Action Mapped keys
Red's Movement
  • Left: A, Q keys
  • Right: D key
  • Gamepad: Left analog stick
Red's Jump
  • Keyboard: W, Z keys
  • Gamepad: Left shoulder button
Red's Action
  • Keyboard: S key
  • Gamepad: Left shoulder trigger
Blue's Movement
  • Left: Left arrow key
  • Right: Right arrow key
  • Gamepad: Right analog stick
Blue's Jump
  • Keyboard: Up arrow key
  • Gamepad: Right shoulder button
Blue's Action
  • Keyboard: Down arrow key
  • Gamepad: Right shoulder trigger

Two players mode

Co-op mode expects at least one gamepad to be connected. If both players will use the keyboard, simply select Single player mode

Red is bound to the first connected gamepad while Blue is mapped either to the keyboard or to the second connected gamepad.

Action Mapped keys
Red's Movement
  • Gamepad: Left analog stick
Red's Jump
  • Gamepad: A button (bottom face button)
Red's Action
  • Gamepad: B, X button (face button next to "A")
Blue's Movement
  • Left: Left arrow key
  • Right: Right arrow key
  • Gamepad: Left analog stick
Blue's Jump
  • Keyboard: X key
  • Gamepad: A button (bottom face button)
Blue's Action
  • Keyboard: C key
  • Gamepad: B, X button (face button next to "A")

Synchronous single player mode

Instantaneous floating platforms anywhere!

This mode is a less demanding alternative to single player mode. Only one character is active at a time and you may switch between them whenever desired.

Action Mapped keys
Movement
  • Left: Left arrow key
  • Right: Right arrow key
  • Gamepad: Left analog stick
Jump
  • Keyboard: X key
  • Gamepad: A button (bottom face button)
Action
  • Keyboard: C key
  • Gamepad: B, X button (face button next to "A")
Switch Active Character
  • Keyboard: TAB, Spacebar keys
  • Gamepad: Y button (top face button)

About custom controls

The game already supports rebind-able controls. Each of the modes described above simply start the game with a different mapping. However, there's still no in-game interface for configuring it. Also, creating a new mapping is quite cumbersome.

If none of the default configurations work for you, do get in contact and we will figure something out!

The red robot

The red robot is an offensive unit. It has more mobility and may use its sword to attack enemies or reflect energy projectiles.

Although the unit's official name is "CRL-000", it's usually simply referred as "Red".

Red is quite fast!

It also may jump really high!

Somehow, it's also able to jump once while in midair!

Some enemy attacks can even be reflected by Red's sword!

The blue robot

The blue robot is a technical unit. It has a teleport gun, which may target either other entities (switching places with them) or surfaces, teleporting the robot to the targeted place.

Although the unit's official name is "RFS-001", it's usually simply referred as "Blue".

Blue is quite slower than Red...

Its jump is also lower...

But Blue's teleport gun make up for all that!

Entities interaction

Entities in this world are usually able to carry each other effortlessly, regardless of their apparent weights.

However, some may be armed with horns, or even weapons. Pay attention!

This kind of interaction should be fun and not broken at all.

About the current title

Back in 2015, I took part of the Global Game Jam and made a small prototype in which a player must control two characters at once.

That prototype was the starting point for a game I made on that same year called JJAT. However, that game feels incomplete, and is too harsh.

This game is meant to be a "spiritual successor" for JJAT. Hence why it's currently being called "JJAT+".

Eventually, the game will be given a proper title. But, until then, I'll keep calling it "JJAT+" (or, sometime "JJAT 2").
